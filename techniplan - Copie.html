Guide rapide

Dessiner le plan
  • Mur : cliquez pour le départ, cliquez pour la fin (angles aimantés à 15°, longueurs en mètres affichées). Échap annule, double-clic termine.
  • Texte : cliquez pour annoter (noms des locaux, repères…).
  • Échelle : 1 carreau = 0,50 m.
Réseaux ⚡ / 💧
  • Choisissez le domaine actif (⚡ Élec ou 💧 Plomb) puis posez des armoires / collecteurs et des équipements (le bandeau sous la barre d'outils propose les types : prises, éclairage, machines… ou vannes, pompes…).
  • Liaison : cliquez une armoire, puis l'équipement qu'elle alimente. Armoire → armoire = définit la provenance.
  • Survolez une armoire : les fils s'affichent vers tout ce qu'elle alimente + infobulle avec provenance et tableau des circuits.
  • Cliquez une armoire (outil Sélection) : panneau avec tableau type Excel des circuits (départ, type, local, C/NC…).
Vérifications C / NC
  • Statut C NC À VÉRIF par armoire et par circuit. Les compteurs en haut suivent l'ensemble.
  • 🖨 Rapport : génère un rapport de vérification imprimable (PDF via l'impression).
Sécurité
  • Sortie de secours : reliable à une armoire (alimentation du bloc) et à sa télécommande. Le survol montre les deux liens.
Divers
  • Molette = zoom · outil ✋ ou clic molette = déplacer la vue · Ctrl+Z = annuler · Suppr = supprimer la sélection.
  • Sauvegarde automatique dans le navigateur. Export / Import pour transférer le fichier .json.
